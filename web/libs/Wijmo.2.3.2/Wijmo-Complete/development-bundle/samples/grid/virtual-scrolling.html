Scrolling

This sample shows how to add a scrollbar to the grid.

  • The scrollMode option determines the scrolling mode in which a scrollbar is added to the grid.
  • The staticRowIndex option indicates the index of data rows that will always be shown on the top when the wijgrid is scrolled vertically.
  • The staticColumnIndex option indicates the index of columns that will always be shown on the left when the grid view scrolled horizontally.
